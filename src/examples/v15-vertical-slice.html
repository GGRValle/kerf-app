V1.5 vertical slice

The full Kerf shell and routes run from the dedicated demo server on port 8010 (path-based navigation). This file is not a second app shell — it only points operators to the canonical entry.

From the repo root:

npm run demo:v15-vertical-slice:serve

Then open http://localhost:8010/dashboard (or /field-capture, /transcript-review, etc.).

npm run demo:v15-slice and demo:v15-slice:serve are aliases for the same 8010 flow.
