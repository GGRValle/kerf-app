F-SA1 · Spec

Command grammar · sales lens

  • One Thing = closest to revenue (proposal to send / deal at risk).
  • On Deck = pipeline next-actions (follow-ups, estimates to finish, site visits, nudges).
  • Pulse = pipeline by stage · win rate · aging · new inbound.

Sales-specific

  • Bottom bar Create = new lead / start estimate. Camera = LiDAR / site measure.
  • Right Hand works the pipeline: flags aging quotes, drafts proposals from the walk, nudges quiet leads.
  • Accent: coral.

Scoped OUT

  • Daily logs · crew time · payroll · other reps' deals.
