F-CHG1 · Spec

Top = project-known

  • Customer + Project # prefill from the project (or GPS geo-locate if on site). Green "From project" chip marks filled fields.
  • RH-assisted banner → opens the blurred one-surface voice overlay (per the v47 fix directive — not a card with buttons). Dictation fills Title / Scope / items.

Shared builder (same as F-EST1)

  • Type = the change-order layout: dropdown of Lump sum · Sections · Itemized. Controls how scope/items render on the client doc.
  • Add item → template picker (Frame 2): Assemblies · Items · Materials · Labor · Subcontractor · Demolition/remodel — pulled from the Cost Library.
  • Totals (Subtotal · Markup · Tax · Discount → Total) auto-calc by permission + settings. Money is integer-cents in storage; nothing posts without review.
  • Preview = client view · Send = the external gate (explicit step, never autonomous).

Applied lesson

  • Back affordance present (the v47 audit found Create/builder surfaces with no mobile back).
Frame 2 · Add item → templates (item detail expands)

From your sketch's "item detail expands → takes you to templates." Add item opens the template categories; picking one drills into the Cost Library and flows selected lines back into Scope with the totals recalculating.
