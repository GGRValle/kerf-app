F-PS1 · Spec

PM lens (blue)

  • One Thing = a job blocker (a selection/decision holding work up).
  • On Deck = missing selections · RFIs · schedule drift · field notes to review · subs to coordinate · client decisions.
  • Pulse = jobs %, budget/margin per job, client state.
  • Sees money + client.

Super lens (cyan)

  • One Thing = today's site problem (crew short, sequencing, quality gate).
  • On Deck = inspections · today's work orders · quality flags · punch · material gaps · crew flags.
  • Pulse = crews/where, site readiness, safety.
  • Sees crew + quality (not margin).

Both

  • Command grammar, same skeleton. Bar: Home · Create · Mic · Camera · More.
PM = coordinationWhat's holding each job up — selections, RFIs, drift, subs, client decisions. The One Thing is always a blocker to clear.
Super = executionWhat's happening on the ground today — crews, work orders, inspections, quality, safety. The One Thing is always today's site problem.
