15.60-second undo state

Canon Operator All operator roles L1–L3 All tenants

After approval, a 60-second undo banner appears. Click to roll back the approval before Platform writes locked. After 60s, lifecycle commits. Corrections after that are new events with correlationId — no Event mutation (CLAUDE.md §3.4).

Source: kerf_wireframes_web_v2.html#frame-undo

KERF Decisions Right Hand · idle CAChristian
Approved · CO-204 sent to Coring · committing in 42s
What undo does
· Reverses approval lifecycle transition (approved → recommended)
· Cancels pending Platform locked write before it commits
· Recalls outbound send if Platform connector supports it (email queue, SMS queue)
· Appends undo event with correlationId to original
After 60s: lifecycle commits. Corrections require a new event chain (no mutation per §3.4).
