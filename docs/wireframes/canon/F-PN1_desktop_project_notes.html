F-PN1 · Spec

Project notebook (D-060 §7)

  • Freeform running context on the project record — decisions, preferences, reminders, gotchas.
  • Written by people or dictated to Right Hand ("jot down that the client prefers matte black").
  • Pinning keeps the load-bearing ones up top.

Distinct from its neighbors

  • Daily Log = dated field capture (what happened on site that day).
  • JobNote = the work artifact from a capture turn.
  • Notes = the persistent notebook that outlives any single day.

Rules carried

  • Back control + sidebar drill-down (Projects → Wegrzyn → Notes), §1/§13.
  • A note can be promoted into a task (F-ON1) or a selection instruction.
