F-RR1 · Spec

4-question rule

  • Macro / comparison / detail? Comparison reference.
  • List / grid / both? Grid-like side-by-side matrix only.
  • Operator cognitive job? Prevent role-root routing drift and make visibility differences explicit.
  • What stays hidden until drill-down? Per-surface copy details and deep nav states.

Why this exists

  • Keeps routing decisions canonical, not ad hoc.
  • Makes accent, home variant, and visibility differences visible in one place.
  • Supports later re-skin and parity work.
What this enforcesEvery role-root uses the same overall grammar, but not the same access. The matrix makes those deltas explicit before they drift into one-off exceptions.
What this refuses to beNot a user-facing screen and not a visual style board. This is internal canon rendered as a working HTML reference.
