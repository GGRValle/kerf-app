F-RT1 v2 · Spec

Matched to the reference

  • Top: 2D / 3D / Walkthrough toggles · Convert to Estimate + Actions/download/print/share.
  • Sub-toolbar: Draw Wall · Measurements · Annotate · Photorealistic.
  • Right: Structures panel, categorized (Walls · Doors · Windows · Cabinetry · Staircases · Openings · Fixtures).
  • Far-right rail: Structures · Products · Source · Upload · Cameras · Elevations.

The Kerf inversion (canon)

  • Reference is tool-first (set Default Wall Type → Draw Wall). Kerf is extraction-first: canvas opens already populated from the scan; Structures shows what Right Hand found (counts + confidence), not an empty palette.
  • The Right Hand banner leads: "read this scan · 38 structures · 4 need your eyes." Tools exist for correction, not authoring from blank. (Houzz-Pro-contrast rule: no surface that asks you to choose a tool first.)

Trust core (non-negotiable)

  • Every measure carries a use-label: Estimate-safe · Verify · Manual — travels into the estimate.
  • Evidence, not truth — you correct + approve before pricing truth.
  • Release gate: cabinetry/stone/glass/millwork stay Verify/Manual until tape/laser logged.
  • Flagged misses (island · partial window) surface explicitly — never silently dropped.

Output + memory

  • Convert to Estimate → Lane-2 Selection/KB contracts (same path), each line with source-scan ref + confidence.
  • RoomRecord persists per (tenant × property/room) — Tenant Room Memory (pre-bid moat).

Naming

  • Room Takeoff / Scan Review, not "LiDAR." Under Projects.
