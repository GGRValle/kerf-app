What changes from F-B1 → F-B2

  • Chrome: sidebar nav stays anchored (all 9 domains visible). No phone-style top "Back" chrome — breadcrumb shows path instead.
  • Layout: two-column body (1.4 : 1 ratio). Left = amount + built-from grid. Right = trust + source + send-gate + audit.
  • Trust banner: expanded by default on desktop. 4 individual checks render inline (source refs / pricing / role visibility / approval) instead of tap-to-expand.
  • Built-from list: 2-up grid of richer cards instead of mobile's stacked list.
  • Amount: larger display (42px vs 32px on mobile).
  • Action footer: Preview neutral-left · Approve primary-right · Edit · Reject · Ask more in middle group. Keyboard hint (⏎) on Approve.
  • Right Hand framing: compressed to one-line synthesis under decision header (mobile dedicates its own card block).
