F-F1r · Spec

4-question rule

  • Macro / comparison / detail? Detail.
  • List / grid / both? Neither. Two-column document comparison only.
  • Operator cognitive job? Verify what was said, edit if wrong, confirm.
  • What stays hidden until drill-down? Original capture metadata, confidence internals, broader project context.

Trust-surface discipline

  • Exception-tier surface reached from draft/source review, not a default workflow stop.
  • No heat maps, no validator stack, no forensic side-panels.
  • Original stays immutable. Working version overlays edits.

Why this is lean

  • The operator already has a reason for opening the surface.
  • The page answers that reason immediately.
  • Audit remains one click away, not spread across the screen.
What this enforces Transcript Review verifies meaning only. It protects the immutable original, shows the current working text, and gives the operator a fast confirmation path.
What this refuses to be Not a confidence console. Not a language-lab screen. Not a daily-use destination. Trust lives in the system; the experience stays light.
