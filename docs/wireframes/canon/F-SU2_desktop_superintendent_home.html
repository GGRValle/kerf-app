F-SU2 · Spec

Role-filtered sidebar (NEW)

  • Primary: Home (active) + Start
  • Speak FAB: cyan-accented (matches role color), separated
  • All · role-filtered: Projects · Schedule · Money (muted · budget only) · Team+Ops (with Crew + Clients sub-items)
  • Sales: NOT visible · Super doesn't access deal pipeline
  • Marketing: NOT visible · Super doesn't access lifecycle outreach
  • Settings only at bottom · no Audit (Super reaches audit via Project Detail when needed)

Money muted treatment

Money sidebar item rendered at 0.45 opacity with "budget only" restrict label. Super has per-project budget visibility (via Project Detail) but no portfolio AR/AP/margin. Visual signal that Money exists in the system but Super's access is scoped.

2-column main layout

  • Left column: One Thing + Today's deployments + Drift to resolve
  • Right column: Quality gates + Field crew comms + Bid-vs-actual variance
  • Desktop adds bid-vs-actual rail (not in mobile due to space) — surfaces the new Schedule-canon quantitative drift agent lane
