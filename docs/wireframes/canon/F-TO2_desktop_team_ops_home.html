Desktop density choices

  • 4-column sub-lens row at desktop width · icon · name+sub-line · state · arrow. Mobile collapsed all into one stack.
  • Sidebar shows 5 sub-lens children expanded when Team & Ops is active · mobile bottom bar can't carry sub-nav, so it lives only in the body.
  • Right rail-led ops synthesis · RH card + Needs-you queue + ops summary + Working-for-you · all telemetry in the rail, list dominates the main column.
  • Ops summary in rail · headcount · open PO total · payroll · reconciliation · QB sync · five quick stats, reference-tier, never page-leading.
  • Working-for-you in rail · mobile carried it inside the phone body below content; desktop puts it in rail so the main column stays focused on the five sub-lens cards.
