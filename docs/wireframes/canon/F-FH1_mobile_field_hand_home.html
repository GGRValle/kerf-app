F-FH1 · Spec

Execution grammar (not command)

  • Clock in → Work Order → Checklist/Capture → Daily → Clock out. No One Thing / On Deck / Pulse.
  • Home is a state machine — reconfigures by clock state (out / in / wrap-up).
  • Main object = today's assigned job (directive, not a decision).

The daily gate

  • Daily must be complete before clock-out — guarantees the log is filed (closes the crew-compliance loop, D-052).
  • Warm, not punitive: "wrap up today," not "you're blocked."
  • Graceful escape: clock out with "missing → super picks it up" — never a dead-end trap.
  • Right Hand pre-fills from the day's capture.

Scoped OUT (permission lattice)

  • No pricing · no margin · no CO state · no client comms · no other crews. Carlos's lane is his lane.

Bar + identity

  • Home · Create · Speak · Camera · More — the fixed five (D-059), bilingual (Inicio · Crear · Habla · Cámara · Más). Speak + Camera stay prominent (voice is the field hand's superpower); Clock in is the home's primary button, not a bar slot.
  • Spanish-native (Mano de Campo). Capture = the shared conversational overlay (F-RH3); SMS ingest (D-052) is the zero-install twin.
