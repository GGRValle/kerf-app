4-question rule

  • Macro / comparison / detail? Comparison.

  • What stays hidden until drill-down? Per-decision source detail, transcript internals, and event-level forensics.

F-AV1b · Grid spec

Grid doctrine

  • Desktop grid exists for weekly macro scan.
  • Right rail disappears in the alternate frame to keep focus on event comparison.
  • Operator still drills into per-project audit from each tile.
What this enforcesAudit comparison belongs in Settings, with list and grid both available because audit events are peer objects worth comparing at scale.
What this refuses to beNot a top-nav command center and not a live-firehose event stream. Trust remains available without overwhelming the operator’s daily flow.
