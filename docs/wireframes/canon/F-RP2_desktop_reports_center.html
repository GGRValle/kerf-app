What this surface proves at desktop density Reports center works as a cross-cutting operational surface without becoming a BI dashboard. Six fixed category cards anchor navigation; recently-run and scheduled lists carry the rhythm. Run · Print · Export on every row makes the printable/exportable promise explicit. RH synthesis + Patterns flagged catch cadence drift ("P&L hasn't been run in 14 days") before the operator has to hunt. Scale holds at 20–50 reports via sectioning, not density.
What this surface refuses to be Not a standalone Reports top-level tab. Not a Buildertrend report builder with 40 filter chips. Not a QuickBooks clone with chart-of-accounts drilldown. Not a Grid view of report tiles. F-RP2 is F-RP1 with breathing room — category grid · wider rows · rail for synthesis and pattern flags. Margin posture stays owner-private; export always audits.
