F-SC1 · Spec

Schedule canon applied

  • Portfolio lens (lightweight) · per-project depth lives in Project Detail Schedule sub-tab
  • No Gantt · no drag-on-grid · list-and-drill
  • Reschedule via action sheet with consequence preview (tap row → "reschedule to..." sheet shows downstream impact)
  • Payment milestones rendered as time-axis events (Money owns records · Schedule renders timing)
  • Celebration milestones tagged for Marketing > Success hand-off

Status chips

  • red · Conflict / drift in time form
  • amber · Payment expected / send trigger
  • green · Celebration milestone (Marketing trigger)
  • blue · Bid-phase visit (Sales origination)

Cross-references

  • Kerf_Layer_A_Subdomain_Map_2026-05-18.md · Schedule canon
  • F-SC2_desktop_schedule_home.html · desktop variant (next)
  • F-PR2/PR4 · per-project Schedule sub-tab (rich task-level)

Export / Print affordances (Lane 7B)

  • Actions strip below RH summary · Print weekly + Export ▾ per-job schedule
  • Print = current week calendar view · Export = selected-job schedule slice (PDF · CSV · XLSX) · audit-logged
  • Audit trail entry on every Export: operator + scope (week or job) + format
