F-MN2 · Spec

Money canon applied

  • Operating money home · action queue not bank statement
  • 4-column rows at desktop · name+meta · scope · state+amount · arrow
  • Margin invisible · separate F-MN3/MN4 (owner-private)
  • QB IIF export operator-triggered · last-export shown · one-way V1
  • Cash pressure callout fires only when triggered · 5-day window
  • Allowances surface here AND in Project Detail Budget · bridge concept

Export / Print affordances (Lane 7B)

  • Top-right bc-bar carries Print + Export ▾ actions alongside New invoice
  • Export format options: PDF · CSV · XLSX (dropdown on click)
  • Audit trail entry written on every Export (data egress · operator + range + format)
  • Print = current view rendered light-mode via existing @media print canon
