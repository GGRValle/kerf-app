  • Cognitive job? Scan 17 events by state × time × source — what Kerf caught, what needed me, who acted
  • Hidden until drill? Validator chain · raw event JSON (lives in F-H1)

Grid anatomy · 17 tiles

  • 2-column mobile · 6 escalated + 10 auto-resolved sample (of 45) + 1 reversed = 17 tiles
  • gt-name = event + project · gt-phase = day + timestamp · gt-chip = state · gt-rhythm = source
  • Red chip = escalated needs decision · amber chip = escalated context-dependent · green chip = auto-resolved · mute chip = reversed
  • Source attribution: Deterministic rule / Right Hand / Operator override (visible in gt-rhythm)
  • `.grid-tile.risk` class not used — Lane 3 doctrine: red is chip-tier only (CSS swept 2026-05-24)

Audit model

  • Settings-bound surface · accessed via More → Settings → Audit (exception-tier, not top nav)
  • Lifecycle states: escalated · auto_resolved · reversed
  • Red is chip-tier only · no full-row red borders
  • Service-trade stress (200+/wk): escalated count rarely exceeds 8-10 · auto-resolved rollup carries volume · pattern holds

Bottom bar (3 + 1 + 1)

  • Mandatory: Home · Projects · Speak (FAB)
