Sub-lens card anatomy

  • Icon tile · color-tinted per sub-lens · scannable signpost
  • Name + chip · sub-lens label · amber chip when action-due
  • State one-liner · plain English · counts + amber accents on action-driving items
  • Left-border accent · amber for attention / behind · transparent for clean
  • Arrow · drill into the sub-lens

Needs you queue

  • Short action queue · 3–5 items max · admin specific (payroll, insurance, reconciliation, invoice signoff)
  • Each row carries the action + dollar/date meta · amber when overdue
  • Sources: timesheet drift, insurance expiry watch, bookkeeping cadence monitor, invoice signoff
  • Maps to the same "Needs you" pattern at Home but scoped to Team & Ops domain

Working for you callout

  • Plain-English statement of background work · trust signal · no jargon
  • Names the watch (timesheet drift · insurance expiry · bookkeeping cadence) without naming agents
  • Quiet visual treatment · right-hand color but soft fill · sits below content not above

What this surface enforces People-axis truth, not employee database. The RH glimpse tells the operator the state of the people-and-admin half of their business in one sentence. The five sub-lens cards each carry a state one-liner so the operator can see which lens needs them today without opening anything. The Needs-you queue concentrates admin work in one column so payroll, insurance, reconciliation and invoice signoff don't get lost across five different screens.
What this surface refuses to be Not an employee database. Not a vendor console. Not a Houzz Pro "Team" page with contact rows and column toggles. Not a time-entry grid. F-TO1 is the people-and-admin glimpse with five drill affordances underneath. Same lens-not-console discipline as F-PR1, scoped to the Team & Ops domain. Individual employees, vendor detail, transaction ledgers stay one click deep.
