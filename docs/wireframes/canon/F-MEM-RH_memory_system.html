Memory helps Right Hand remember. The graph tells Right Hand what is true.

◈ The synapse — how a query reaches truth without reading everything

Query / conversation
"what's the schedule state?"
→
Tier-0 triage
discerns what's needed + which route to fire
→
Topology map
holds routes, not data — "schedule lane → here"
→
Retrieval (L3)
pulls only the relevant slice
→
Graph (L2)
confirms what's true · commits
The orchestrator holds a lightweight map (routes, not archive) so working memory stays small and fast. Cost scales with what the query needs — not with total tenant history. The pile grows forever; the per-query reach stays cheap. Modality-matched: shape questions return visual slices, fact questions return structured records. Compaction is this same mechanism on the time axis: demotion-with-a-route, never summarization-with-loss — when context fills, detail demotes a tier and leaves a breadcrumb; nothing is destroyed. New eval: after compaction, every demoted load-bearing fact must still be reachable (route completeness).

The grand architecture is the generalization of what #305 already proved — not a new system.

The floors — enforced at every layer

▎
Tenant isolation. No cross-tenant recall, ever. Structure may promote platform-wide (Sentry-governed); tenant facts never cross the wall.
▎
Source support. Every committed fact traces to a ledger source ref.
▎
No durable action without confirm. The gate. Memory proposes; the human commits consequential / financial / contractual actions.
▎
No raw model output as truth. Server validates, floors, merges before commit.
▎
Deterministic validation at the write path — not model discretion.

Core evals — failure modes are regressions, not edge cases

●
Tenant bleed — tenant A's fact surfacing for tenant B. Zero tolerance.
●
Wrong-job recall — Chen data attributed to Wegrzyn (the live failure already seen).
●
Stale scope — recalling superseded scope as current.
●
False closure — "we already filed it" with no ledger event. The honesty floor as a test.
●
Fabrication accumulation — the curator inventing facts during promotion.
●
Omission — a load-bearing fact dropped in compaction (the original scar).
●
Route loss — a demoted fact whose breadcrumb wasn't written or points wrong: silent omission one layer over. Compact → probe → a miss is a regression.

The strategic line — why this is safe to adopt rather than build

Buy the plumbing; build the judgment. The moat is not "Right Hand has vector memory" — the substrate is commoditizing for everyone. The moat is the judgment riding on top: contractor-specific salience · tenant-safe learning · multimodal site-evidence distillation · "what matters next / what to interrupt Christian about" · consequence & money gates · wrong-job risk · scope-drift detection · Sentry-reviewed Blackboard plays · per-tenant CRAFT. The substrate retrieves; the graph commits; Sentry governs; the judgment is ours.
