F-AA1 · The Attention Artifact — one card, three mounts, five states

The render target for the live Attention Manager (D-053 two-artifact rule · sibling to Lane B's JobNote work-artifact). One component renders the same object in three places — Home (One Thing · On Deck · Pulse), the Right Hand overlay's "what's next," and the relay / Office-Review queue. Stage A (built) projects these from real events, deterministic order, honest empty state. Stage B (next) makes the headline model-written and the rank + per-role projection model-led — rendered into this same card. Build Stage B to this card.

The contract: ① one component, props only — Home/overlay/relay pass layout, not different cards · ② five states drive the left bar + pill: needs_you · handled · next_options · risk_changed · review_suggested · ③ every card carries domain · headline · because · consequence-tier · source · expand · ④ honesty floor — durable state comes from the work artifact/TRP, never model copy ("captured," not "filed," until the write returns) · ⑤ red is chip/bar only, never a full-row alarm · ⑥ no events → honest empty state, never fixtures.

Anatomy (every card)

  • Left bar + state pill — one of the five states (color-coded).
  • Domain dot + tag — which of the 9 domains (Wegrzyn=project, Money, Team, Selections…).
  • Headline — what it is. Stage A: derived. Stage B: model-written.
  • Because — why it needs you (event-derived); honest ("captured," not "filed").
  • Consequence chip — reversible / durable / money / send (a label; the action stays gated on its own surface).
  • Source chip (via voice/photo/text) + expand → the underlying surface + linked work artifact.

Stage A (built — what's true now)

  • One component; Home / overlay / relay render it from the same attention list (`attentionArtifact.ts`).
  • Projected from real events; deterministic order (recency · consequence · domain); honest empty state, no fixtures.
  • Durable state from the work artifact/TRP, not model copy.
  • Relay/Office-Review is a projection, not a new domain.

Stage B (build to this card)

  • Headline + because are model-written ("RIGHT HAND SAYS…"), governed by the validator wall (no money written, no autonomous send, every claim has a source_ref).
  • Rank is model-led — the One Thing is the top-ranked artifact, On Deck the ranked queue, Pulse the ambient tail.
  • Per-role projection — same graph, ranked/filtered per role-root (Owner=decisions · Field=today's work · Admin=risk/deadlines).
  • Renders into this card — don't invent a new layout. Honesty floor + red-is-chip unchanged.
