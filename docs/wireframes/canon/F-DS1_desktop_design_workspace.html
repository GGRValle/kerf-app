F-DS1 · Spec

Design under Sales (D-060 §2)

  • Estimator/sales seat. Five tabs: Mood board · 3D model · Selections · Design tools · Integrations.
  • The creative workspace that produces selections — it doesn't replace the Selections entity, it feeds it.

Ties to the Knowledge Base

  • "Pull from Library" drops Selections Library items onto the board.
  • Approve → Project Selection instance (PO + allowance + schedule). New picks save back to the Library.
  • Assemblies/templates can seed a board.

The other tabs

  • 3D model = RoomPlan/LiDAR capture (use-labels apply).
  • Integrations = Bluebeam (2D plan markup) + the rest, per the Connections registry.

Rules carried

  • Back control (non-home, §1). "Save to estimate" is the bridge to the estimate builder.
