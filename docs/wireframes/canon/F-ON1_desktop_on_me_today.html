F-ON1 · Spec

Two streams, one surface (D-060 §8)

  • Right Hand surfaced = generated Attention Artifacts (the D-053 Attention Manager). Five states, same card as Home.
  • Your list = user-authored tasks. Not generated — written by you or dictated to Right Hand.
  • No second inbox: this is the single "what's on me."

Authored tasks

  • Personal (global to you) or project-scoped (carries a project chip).
  • Task + subtask nesting.
  • Reconcile with the Schedule — a project task can post to the schedule; a scheduled item can surface here. schedule_ref.
  • Toggle: All · This project · Personal.

Bridges

  • A surfaced Attention item → "+ Add to my list" makes an authored task linked to it.
  • Still a projection over the 9-domain graph, not a 10th domain.

Rules carried

  • Back control because it's a non-home surface (D-060 §1).
  • Universal — every role-home links here.
