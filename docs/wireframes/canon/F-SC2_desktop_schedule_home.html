F-SC2 · Spec

Schedule canon applied

  • Portfolio lens · per-project depth lives in Project Detail Schedule sub-tab
  • No Gantt · list-and-drill only
  • Payment milestones rendered as time-axis events (Money owns records)
  • Celebration milestones tagged for Marketing > Success hand-off

Right rail composition

  • RH synthesis · cyan-of-the-day if a conflict fires
  • At-risk callout · the single conflict to resolve today (red-tinted card · scoped not row-wide)
  • This-week summary · deployments · payments · celebrations · conflicts · holding

Cross-references

  • F-SC1_mobile_schedule_home.html · mobile counterpart
  • Kerf_Layer_A_Subdomain_Map_2026-05-18.md · Schedule canon

Export / Print affordances (Lane 7B)

  • Top-right bc-bar carries Print weekly + Export per-job ▾ alongside the timestamp
