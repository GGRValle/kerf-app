F-RH4 · Spec

The composer

  • + (left) opens "Bring something in": Photo · File/PDF · Paste · Scan · From a project — the visible invitation to import. Same + on the mobile composer.
  • Grows with what you type/dictate (caps then scrolls); conversation slides up. Voice + type + paste coexist.
  • Placeholder reads Ask, or drop a file… — drag-drop and ⌘V paste land here too, not just the +.

Review-a-document (warm input)

  • An attachment is a chip in the conversation (via upload / via paste); it also files as a job document.
  • Right Hand reviews — extracts scope, dims, gap-flags — reversible, no confirm.
  • Turning it into an estimate/CO/note is the consequence point (the green bubble) — "Start estimate · Keep on file · Keep talking."

Trust (unchanged)

  • Evidence, not truth — RH proposes, never auto-acts. Tenant-scoped, validator-walled, no model-written money, source refs on every claim.

Cross-surface

  • Same contract as F-RH3 (⑥ grows · ⑦ paste/attach). Desktop = right-docked panel; mobile = overlay. The + is on both.
