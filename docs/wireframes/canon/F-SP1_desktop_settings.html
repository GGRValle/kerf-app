F-SP1 · Spec

4-question rule

  • Detail — configuration landing, not operational work queue
  • List only — card grid; Grid view: NO
  • Cognitive job — configure account, permissions, plays, integrations, audit access, bar customization
  • Hidden until drill — full audit stream, play editor depth, billing checkout, per-integration OAuth flows

Navigation canon

  • Settings active in sidebar · All domains collapsed (no expanded sub-items)
  • Account card drills to F-SP1a — username · password · avatar · notifications · language
  • No Audit in top nav — Audit card drills to F-AV1b
  • Customize navigation drills to F-BC1 (mobile bar picker)
  • Billing: V1 Solo stub only
EnforcesAudit deep-link policy — audit access is settings-bound. Navigation customization is discoverable without cluttering Home.
RefusesNot a system admin console. Not a billing checkout. Not a grid of users. Grid view: NO.
