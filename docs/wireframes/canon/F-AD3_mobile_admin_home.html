F-AD3 · Spec

Command grammar · admin variant

  • One Thing = a risk or deadline (insurance, payroll, expired COI), not a decision.
  • On Deck = the open stacks (AR/AP, sub docs, POs, rebates, scheduling holes).
  • Pulse = a denser status board — money, subs, team/HR, rebates. The one role where dashboard density is correct.

Sentry-forward

  • Compliance watch (OSHA/IIPP/CSLB · insurance · certs · COIs) gets real estate above the fold — admin is the role whose job is "what's about to bite us."
  • Rebates/programs are a worked lane (Würth-style) — Right Hand flags unclaimed + thresholds.

Bar (the fixed five)

  • Home · Create · Speak · Camera · More — the same bar as every role (D-059). Admin's distinctiveness is in the content (Sentry above the fold, Money/compliance status board) and in More, not the bar. Magenta accent. Audit stays owner-private even from admin.
Always a risk or a deadlineAdmin's One Thing is never strategy — it's the thing that bites if it slips: insurance, payroll, an expired COI. Sentry rides up top because compliance is the admin's lane.
Density is correct hereWe kill dashboards everywhere else, but admin genuinely monitors many lanes at once — so the Pulse is a real status board. The bar stays the standard five; the density lives in the content, not the navigation.
