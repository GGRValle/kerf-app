F-ES4 · Spec

4-question rule

  • Macro / glimpse — sales motion summary for Jamie, not a pipeline grid
  • List only — On Deck rows and rail lists; Grid view: NO
  • Cognitive job — what deals need me today? estimates due? where are leads stalling?
  • Hidden until drill — margin posture, cash, owner strategy, full pipeline table, Audit

Sidebar IA (Estimator)

  • Home (active) · Start · SPEAK (violet FAB)
  • Sales expanded: Pipeline · Proposals · Deals
  • Projects limited: My assigned only
  • Schedule limited (no sub-items)
  • Marketing expanded · Attribution active sub
  • Settings — no Audit in nav

Main layout

  • 3 columns: One Thing · On Deck · The Pulse
  • Right rail: 4-cell attribution rollup · top stalled leads · estimates due this week
  • Red chip-tier only (e.g. stale on Ramirez)
